# Usage Policy

## Intended Use

This software is designed for **individual developers** to analyze their own Claude Code usage patterns, gain personal productivity insights, and improve their workflows.

## Prohibited Uses

This software **shall not** be used for:

- **Employee monitoring or surveillance** — Corporations, managers, or organizations must not use this tool to track, evaluate, or monitor employee productivity, coding behavior, or work patterns.
- **Performance evaluation** — Output from this tool must not be used as input for employee performance reviews, ranking systems, or disciplinary actions.
- **Workplace surveillance infrastructure** — This tool must not be integrated into systems designed to surveil or profile workers.

## Scope

This policy is advisory and reflects the ethical intent of the project maintainers. The MIT License governs the legal terms of use. However, contributions that enable employee monitoring use cases will not be accepted.

## Reporting Misuse

If you become aware of this software being used for employee surveillance, please open an issue on this repository.
